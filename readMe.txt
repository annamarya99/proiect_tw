Web application for bug tracking

The web application, developed using Node.js and React, serves as a bug tracking and communication platform for project teams. It facilitates effective communication among team members regarding the resolution of bugs within an application. The application leverages a SQLite database for efficient data storage. Its user-friendly interface and collaborative features enable seamless interaction and coordination among team members.
